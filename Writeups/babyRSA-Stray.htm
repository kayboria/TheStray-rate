Solution

They are giving a very small e factor so we can do Broadcast Attack in RSA, here is the attack description

"The Broadcast Attack in RSA encryption arises when the same plaintext message is encrypted with the same small public exponent but different moduli for different recipients. This scenario allows an attacker to decrypt the message without needing the private key, exploiting weaknesses inherent in the use of small exponents and the lack of proper padding."

Now, applying the attack is very easy, we have e is so small which is 37, and so is the two arrays above 37 element in the ciphertexts collected and 37 element in the moduli array. Then we use CRT The Chinese Remainder Theorem on these pairs to compute a combined integer that satisfies all the congruence relations of ciphertexts and moduli and get the 37th root so the steps are as follows:

  • Check if the length of c and n are the same and equal to the exponent
  • Using Chinese Remainder Theorem (CRT) to combine the ciphertexts
  • Taking the 37th root of the combined result
  • Convert the number back to bytes and then to a string

if len(c) != 37 or len(n) != 37:
